## ğŸš€ Getting Started

### Prerequisites

- Node.js (v20.x or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd hr_web
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:5173`
